This package provides shared browser components for the Zope Toolkit.
These components consist of a set of interfaces defining common
concepts, including:

- ``IView``
- ``IBrowserView``
- ``IAdding``
- ``ITerms``
- ``ISystemErrorView``

Documentation is hosted at https://zopebrowser.readthedocs.io
